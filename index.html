Storm Tutorial

Storm System

Process streams of data

  • Distributed
  • Reliable
  • Fault-tolerant

Storm Basics

  • Spout : handles input stream of a storm cluster
  • Bolt : acquires data to process from spout

Example Code

builder.setSpout("spout", new RandomSentenceSpout(), 5);
builder.setBolt("split", new SplitSentence(), 8).shuffleGrouping("spout");
  • Input stream sends data to spout (here named spout)
  • Bolt acquires the data from spout (here bold named split)
  • Do not worry about syntax which will be explained later
